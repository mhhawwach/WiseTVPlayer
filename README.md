## Install

**On a phone / tablet**
1. Download `WiseTVPlayer-v1.0.2.apk` to your device.
2. Open it and allow installation from this source if prompted.

**On Android TV / Fire TV**
1. Enable **Apps from unknown sources** (Settings → Device → Developer options, or Fire TV: Settings → My Fire TV → Developer Options).
2. Use a sideload app (e.g. *Downloader* / *Send Files to TV*) to transfer and install the APK.

## Features

- **All your content in one place** — Live TV, Movies and TV Series, neatly organized
- **Premium playback** — resume where you left off, auto-play next episode, multiple audio tracks & subtitles, Picture-in-Picture, adjustable picture fit
- **Live TV done right** — Catch-Up TV, on-screen resolution badge, favourites
- **Find anything fast** — universal search across live, movies and series
- **Personal & secure** — multiple profiles, multiple server connections, parental PIN controls, customizable categories
- **Made your way** — themes & wallpapers, text-size options, full English & Arabic (RTL) support
- **Works everywhere** — phones, Android TV, Fire TV and Samsung TV
